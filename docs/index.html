Custom Parameters

Edit the parameters in main.py to customize your transformation:

main(
    brighness_weight=1.0,        # Weight for brightness matching
    frequency_weight=1.0,        # Weight for frequency matching
    distance_weight=0.01,        # Weight for spatial distance
    stiffness=0.005,             # How quick each cell moves
    damping=0.98,                # How quick each cell slows down
    stretch_factor=0.1,          # Cell stretch when accelerated
    source_path="demo/imgs/random_noise.png",
    target_path="demo/imgs/kirk.png",
    cell_size=1,                 # Grid cell size (smaller = more detail)
    quick_gen=False              # Fast generation mode
)

Parameter Reference

Parameter Description
stiffness Controls how quickly cells move towards their target position. Lower values create slower, more fluid motion.
damping Controls how quickly cells slow down. Higher values (closer to 1.0) create smoother motion.
stretch_factor Controls how much cells stretch when accelerating. Higher values create more dramatic stretch effects.
cell_size Size of grid cells in pixels. Smaller values = more detail but slower processing.
quick_gen Enable fast generation mode with lower quality but faster processing.
source_path Path to the source image file.
target_path Path to the target image file.

üéØ How It Works

  1. Grid Splitting: Both source and target images are divided into a grid of cells
  2. Feature Extraction: Each cell's brightness and frequency characteristics are analyzed
  3. Matching: Source cells are matched to target cells based on visual similarity and spatial distance
  4. Physics Simulation: Cells move to their target positions using spring physics
  5. Animation: The transformation is rendered frame-by-frame into a smooth animation
